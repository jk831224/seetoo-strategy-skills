# 分析報告範本

固定區段對應 core `05` 十步驟。全程用三分區標籤：
**【事實】**（使用者提供）／**【框架】**（來自 core，附出處）／**【判斷】**（分析推論，標〔推測〕）。

## 使用提醒
- 「框架」欄/句必附 core 出處（檔名＋章節）。
- 任何對該事業的推論一律落在【判斷】並標〔推測〕；**不要**把推論寫進【事實】。
- 形態速寫模式：只出步驟 1–3（報告第 0–3 區段）＋待問清單。
