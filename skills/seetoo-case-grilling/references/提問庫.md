# 提問庫（追問彈藥）

依司徒達賢策略概念分類；每題都扣回 core。**用問的，不要替對方作答。**

## 1. 分辨事實 vs 推測（最先用、最常用）
- 這是你**親眼觀察到的事實**，還是你的**推測/期望**？
- 你**怎麼知道**？有沒有反例？
- 這是「目前確實如此」，還是「你希望未來如此」？
> 對應 core `02`：策略構想中的「配合呼應」在過去成功裡是既存事實，在未來則是待驗證前提。

## 2. 六大構面探問（產市垂規地競，逼具體）　〔core `01`〕
- **產**：產品線多寬？特色到底是什麼？「品質好」——好在哪、怎麼讓客人看得到？
- **市**：客戶到底是誰？你**不服務**哪一塊？怎麼區隔？
- **垂**：哪些自己做、哪些外包？為什麼是這條線？
- **規**：相對主要對手規模如何？規模經濟出現在哪個環節？
- **地**：為什麼是這些地區/通路？
- **競**：客戶為什麼選你不選對手？這個優勢**難複製**嗎？多久會被追上？

## 3. 逼出並反證前提　〔core `02`〕
- 這個策略要成立，背後的**前提**是什麼？（環境前提？條件前提？目標前提？）
- 這些前提是**既存事實**還是**待驗證假設**？你驗證了嗎、怎麼驗證？
- 如果**這個前提不成立**，你的策略會怎樣？（逼到「曲高和寡就收攤」這種具體後果）

## 4. 因果關係壓力測試　〔core `10`/`07`/`08`〕
- 你說「A → B」——中間**少了哪一步**？真的有因果，還是只是相關？
- 這條因果鏈要持續成立，需要誰持續做什麼？哪個環節最脆弱？
- 哪些單位若各自求效率，會**破壞**這條因果鏈？（core `10` §G／`08` 警示）

## 5. 非答案偵測（聽到就退件）　〔core `10` §E〕
- 「運氣好／時機好」→ 那**別人也碰到同樣時機，為什麼是你成？** 具體做了什麼？
- 「我們很努力／團隊很強」→ 努力**在哪個構面**展現成具體做法？換成對手努力就贏不了你嗎？

## 6. 反主張攔截（導回方法）　〔core `09`〕
- 對方做 **SWOT** 起手 → 「還沒有具體策略方案，怎麼判斷某項是優勢或劣勢？先講策略形態。」
- 對方從 **mission/vision** 起手 → 「願景是文宣；先講你**實際**的六大構面與前提。」
- 對方從**環境分析/設定目標**起手 → 「先有具體方案，再深入驗證環條目；目標是限制條件，不是起點。」

## 7. 道理挑戰（進階）　〔core `07`/`08`〕
- 規模經濟/範疇經濟在你這真的存在嗎，還是規模不經濟？
- 這個價值活動該**自製還是外包/內部化**？交易成本怎麼算？
- 這是**配適(fit)**還是硬湊？你的能力與資源撐得起這個形態嗎？

## 收斂用
- 我們釐清了：你的策略形態是…；關鍵**待驗證前提**是…；最脆弱的因果是…。
- 你接下來要去**驗證**哪一個前提？怎麼驗證？
